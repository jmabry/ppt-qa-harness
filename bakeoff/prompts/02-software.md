# Prompt 2: Software Development

Create a 10-slide technical presentation about "Migrating from Monolith to Microservices" for a team of 20 backend engineers. Include:

- Cover slide with the talk title and speaker info (use placeholder names)
- Current state: the monolith's pain points (deployment bottleneck, blast radius, team coupling)
- Target architecture: service boundaries, API gateway, event bus
- Migration strategy: strangler fig pattern with 4 phases
- Risk matrix: what can go wrong and mitigations
- Timeline: 6-month phased rollout
- Monitoring & observability: what to instrument from day one
- Team structure: how to organize squads around services
- Success metrics: latency, deployment frequency, MTTR
- Q&A / closing slide

Use a clean, technical aesthetic. Dark mode preferred. Include diagrams where possible (layered boxes, flow arrows).
